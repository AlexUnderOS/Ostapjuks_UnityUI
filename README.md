# Little guide

### Start
1. Press start button

![start_btn](readme_/menu.png)


### Character selection

Fill in the input fields, for example:
1. name `Alex`
2. password `2006`
3. Select a character skin for customization
4. Click the create button

![start_btn](readme_/newCharacter.png)
